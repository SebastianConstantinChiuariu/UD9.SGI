Solucionari Visual Studio Code

  1. Instal·lació Visual Studio Code

    Per instal·lar VSCode amkb el teu navegador cal cercar "Visual Studio Code" i accedir al primer enllaç, a la pàgina inicial apreixerà el instalable.

  2. Iniciar Sessió a VSCode

    Al panell esquerrà, és situa l'opció Comptes i activarem el compte de GitHub, es sincronitzarán tots els canvis de l'aplicació.

  3. Extensions

    Al panell esquerrà, és situa l'opció Extensions on haurà tot tipus d'extensions estètiques i d'editor de codi, com pot ser Python, HTML o C++.

  4. Creació de Carpetes i Fitxers

    Per crear carpetes i fitxers, has d'anar a la part superior del programa, allà es troba un panell que incorpora les funcions per crear-les, aquestes és diferencien pel seu dibuix.

  5. Errors de Sintaxi

    Els errors de Sintaxi són errors de codi font que es forma per una desconcordança. Revisa l'estructura del codi. En cas que no funcioni, s'hauràn de reiniciar les extensions de codi, ja que aquests podrien generar una fallada o que no estiguin habilitades.

  6. Importació de Llibreries

    Visual Studio Code permet importar llibreries, per tant es poden importar noves funcions i eines gràfiques, permet crear entorns gràfics, eines de scrapping o jocs simples. Per import una llibreria cal inserir 'import NOMLLIBRERIA'

  7. Actualitzar VSCode

    De vegades, el programa notificarà que hi ha una actualizació disponible, caldrà actualitzar per obtenir la nova versió millorada de VSC. Per actualitzar el programa, ens inserirem a

  8. Jerarquía del Codi

    A l'hora d'escriure el codi font, s'han de respectar la estructura del que es composa, la composició del codi ha de contenir tabulacions per definir la jerarquía del projecte.

  9. Llibreries gràfiques

    Una llibreria gràfica molt popular al llenguatge Python, es Tkinter. Aquesta llibreria es composa de distints elements gràfics com ara butons, frames, caselles entre altres elements gràfics.

  10. Guardar el programa

    Per guardar el programa, t'has de situar a la part superior del programa, i a la pestanya 'Arxiu', allà sortirà 'Guardar com a' i selecciones el directori a on vols que es guardi.

Arà hauràs après les preguents més frequents que sorgeixen a l'hora d'utilitzar el programa, esper que hagi t'hagi ajudat, Bon dia :).
